# 光猫精灵 - 扫码登录路由器APK

## 项目简介

光猫精灵是一个Android应用，用于通过扫描二维码快速登录路由器管理界面。

## 功能特点

- 扫描路由器二维码快速登录
- 手动输入路由器信息登录
- 检查网络连接状态
- 支持常见路由器登录路径

## 安装说明

### 方式一：直接安装APK
1. 下载 `app/build/outputs/apk/release/app-release.apk` 文件
2. 在Android设备上安装该APK文件
3. 打开应用即可使用

### 方式二：从源码构建
1. 确保已安装Android Studio
2. 打开 `android` 目录作为项目
3. 构建并运行应用

## 使用说明

### 1. 生成路由器二维码

使用 `generate_qr.py` 脚本生成包含路由器信息的二维码：

```bash
python generate_qr.py
```

在弹出的界面中：
- 输入路由器IP地址（如：192.168.1.1）
- 输入登录用户名（如：admin）
- 输入登录密码（如：admin）
- 点击"生成二维码"按钮
- 点击"保存二维码"按钮保存二维码图片

### 2. 扫码登录

1. 打开光猫精灵应用
2. 点击"扫描二维码"按钮
3. 扫描上一步生成的二维码
4. 应用会自动填充登录信息并尝试登录
5. 登录结果会显示在界面上

### 3. 手动登录

1. 打开光猫精灵应用
2. 手动输入路由器IP、用户名和密码
3. 点击"手动登录"按钮
4. 登录结果会显示在界面上

### 4. 检查网络连接

点击"检查网络连接"按钮，应用会显示当前网络连接状态。

## 技术实现

- 使用 Kotlin 开发
- 使用 ZXing 库实现二维码扫描
- 使用 OkHttp 库实现网络请求
- 使用 PermissionX 库处理权限请求

## 注意事项

1. 确保手机与路由器在同一局域网内
2. 不同品牌的路由器登录路径可能不同，默认使用 `/login.cgi`
3. 如登录失败，请检查路由器的实际登录路径并修改代码
4. 扫码登录功能需要相机权限

## 支持的路由器

- 华为路由器
- 中兴路由器
- TP-Link路由器
- 水星路由器
- 其他使用标准登录路径的路由器
